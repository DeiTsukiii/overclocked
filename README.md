# 🌐 Overclocked: Modular Data Automation

An immersive, node-based automation and incremental simulation game built with **Phaser.js**. Step into the shoes of a rogue data broker and systems architect. Buy advanced hardware modules, wire them together dynamically, download massive text and image datasets, filter out malicious viruses, and sell pristine data on the digital black market to scale your operation from a local $200 setup into a multi-billion dollar server empire.

---

## 🚀 Core Features

- **Dynamic Wiring & Node-Graph System:** Create intricate pipelines by dragging visual connections (wires) between component output and input ports with custom type safety.
- **Robust Machine Catalog:** Deploy 11 unique types of operational modules across 4 classes (Resources, Downloaders, Security, and Money generation).
- **Infinite Hardware Upgrades:** Level up critical modules (Networks, CPU Cores, and Coolers) to exponentially multiply your processing throughput and data generation capacity.
- **Malware Filtration Loop:** Encounter real-time security threats. Intercept incoming viruses using Antivirus Filters and turn a profit off malicious files using Quarantine Walls.
- **Advanced Performance Tweaks:** Manually overcharge your data factories with Overclockers while counterbalancing high thermal ceilings using Liquid Coolers.
- **Guided Progression Engine:** Features a reactive multi-tier objective system tracking 20+ tasks that guide you from network deployment to the $1,000,000,000 milestones.
- **Immersive RTS Sandbox Controls:** Effortlessly navigate an expansive 200x200 grid environment with seamless mouse-wheel zooming, right-click panning, and intuitive item dragging.
- **Persistent Local Automation:** Includes fully integrated local storage save states, ensuring your massive network configuration is retained when returning to the game.

## 🔌 Signal Wiring & Port Guide

Every module communicates via color-coded, type-safe data ports. Wires can only link an **Output Port** to a matching **Input Port**.

* 🟩 **Download (`0x00ff88`):** Bandwidth allocation for data ingestion pipelines.
* 🟦 **Upload (`0x00bfff`):** Bandwidth allocation for payload distribution and item selling.
* 💎 **Text Data (`0x00ffcc`):** Extracted light data files (100 KB base size).
* 🔮 **Image Data (`0xff00ff`):** Extracted heavy assets (1.5 MB base size).
* 🟡 **Money (`0xccff00`):** Financial asset pipelines directing wealth toward automated banks or collectors.
* 🔥 **Clock Speed (`0xff6600`):** Computing clock frequencies generated by processing hardware.
* ❄️ **Coolant (`0x00ffff`):** Cryogenic fluid flow used to combat catastrophic overclocker thermal overloads.
* 🛑 **Virus (`0xef4444`):** Extracted malicious vectors that must be safely isolated or sanitized.
* ⚪ **Universal Empty (`0xffffff`):** Adaptive placeholder ports that dynamically lock to matching data types upon first wire connection.

## 🕹 Controls & Orchestration

- **Pan Camera:** Hold `Right-Click` and drag to glide seamlessly around your network grid space.
- **Zoom Canvas:** Scroll the `Mouse Wheel` up or down to alternate between micro-optimization views and bird's-eye architectural layouts.
- **Move Module:** Hold `Left-Click` over a component's center body to drag and snap it onto the $40\text{px}$ alignment grid.
- **Establish Wire Connection:** Click an output port node once with `Left-Click`, then click an open, color-compatible input port node to complete the wire.
- **Sever Wire Connection:** `Double-click` or tap any port node rapidly within $300\text{ms}$ to instantly disconnect all incoming or outgoing lines attached to it.
- **Decommission Hardware:** Click the red **`X`** in the top-right corner of any module to delete it and recoup a $50\%$ scrap refund.
